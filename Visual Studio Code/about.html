CSS

The second language I had to learn to make this website was CSS.
    CSS stands for Cascading Style Sheets.
    The Boston University article What Is CSS? describes it like this: 
    "CSS specifies your document's style, page layouts, colors, and fonts."
    It interacts with the HTML you already have through selectors.
    The categories of CSS selectors include:
  • Simple selectors (select elements based on name, id, class).
  • Combinator selectors (select elements based on a specific relationship between them).
  • Pseudo-class selectors (select elements based on a certain state).
  • Pseudo-elements selectors (select and style a part of an element).
  • Attribute selectors (select elements based on an attribute or attribute value).
    • Once I knew how to select an element, I began to use declarations to style the webpage.

    JavaScript
    The final programming language I had to learn to make this website is called JavaScript.
    This is the language that makes the math calculator work.
    According to the Amazon Web Services article What Is JavaScript? 
    "JavaScript is a programming language that developers use to make interactive webpages."
    One way it can do this is by taking inputs from the user to deliver an output that 
    is shown on the website. The way that JavaScript tells the computer what to do is through 
    the use of statements. There are five kinds of statements in JavaScript: 
    Values, Operators, Expressions, Keywords, and Comments.
    Most often, JavaScript uses keywords at the beginning to identify what action to perform.
